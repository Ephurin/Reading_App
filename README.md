## Prerequisites

Before you begin, ensure you have the following installed on your system:

- **Android Studio** (Latest version recommended)
  - Download from: https://developer.android.com/studio
- **JDK 11 or higher**
  - Android Studio usually includes JDK
- **Git**
  - Download from: https://git-scm.com/downloads
- **Minimum Android SDK**: API 21 (Android 5.0) or higher

## Getting Started

Follow these steps to download and run the Reading App on your local machine.

### 1. Clone the Repository

Open your terminal or command prompt and run:

```bash
git clone https://github.com/Ephurin/Reading_App.git
```

Or download the ZIP file from GitHub and extract it.

### 2. Open the Project in Android Studio

1. Launch **Android Studio**
2. Click on **"Open"** or **"Open an Existing Project"**
3. Navigate to the cloned `Reading_App` folder and select it
4. Click **"OK"**
5. Wait for Android Studio to sync and build the project (this may take a few minutes)

### 3. Configure the Project

#### Sync Gradle Files
- Android Studio should automatically sync Gradle files
- If not, click on **File > Sync Project with Gradle Files**

#### Update local.properties (if needed)
- This file is auto-generated by Android Studio
- It contains the path to your Android SDK
- You don't need to create it manually

### 4. Run the Application

#### Option A: Run on an Android Emulator

1. Click on **Tools > Device Manager** (or AVD Manager)
2. Create a new virtual device if you don't have one:
   - Click **"Create Device"**
   - Select a device (e.g., Pixel 6)
   - Choose a system image (e.g., Android 13 or higher)
   - Click **"Finish"**
3. Click the **Run** button (green play icon) in Android Studio
4. Select your emulator from the device list
5. Wait for the app to install and launch

#### Option B: Run on a Physical Device

1. Enable **Developer Options** on your Android device:
   - Go to **Settings > About Phone**
   - Tap **Build Number** 7 times
2. Enable **USB Debugging**:
   - Go to **Settings > System > Developer Options**
   - Turn on **USB Debugging**
3. Connect your device to your computer via USB
4. Click the **Run** button in Android Studio
5. Select your device from the list
6. The app will install and launch on your device

### 5. Build APK (Optional)

To create an installable APK file:

1. In Android Studio, go to **Build > Build Bundle(s) / APK(s) > Build APK(s)**
2. Wait for the build to complete
3. Click **"locate"** in the notification to find your APK
4. The APK will be in: `app/build/outputs/apk/debug/app-debug.apk`

## Troubleshooting

### Gradle Sync Failed
- Check your internet connection
- Go to **File > Invalidate Caches / Restart**
- Try **File > Sync Project with Gradle Files**

### SDK Not Found
- Go to **File > Project Structure > SDK Location**
- Ensure Android SDK path is correctly set
- Download missing SDK components via **Tools > SDK Manager**

### Build Errors
- Clean the project: **Build > Clean Project**
- Rebuild: **Build > Rebuild Project**
- Check that you're using compatible Gradle and Android Studio versions

### Emulator Not Starting
- Ensure virtualization is enabled in your BIOS
- Allocate more RAM to the emulator in AVD settings
- Try creating a new virtual device
